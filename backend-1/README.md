## Usage

To start the server, run:
```
npm start
```

The server will be running on `http://localhost:3000`.

## API Endpoints

- **GET /numbers**: Retrieve all number entries.
- **POST /numbers**: Create a new number entry.

## Models

### Number Model

- **numberPlate**: String - The plate number of the vehicle.
- **timestamp**: Date - The time when the entry was created.
